Practice Lesson

Let's Make a Website

1. Starting Your Website

1. Create a new pen through codepen. Codepen is a website which allows you to write code in the browser, and see the results of it as you build.

2. Name your website. It can be your name or anything relevant to your website contents. Click the pen on the top left hand corner to name your website. Example: "Madeleine's Website"

2. Create Your Website Through HTML

HTML stands for HyperText Markup Language. HTML displays all of your text, pictures, videos and more!

1. Start by creating the title of your page by using the "h1" tags. An "h1" tag is normally used for the title or any emphasized text that you want on the page. "h1" is the largest header tag you can have and the smallest header tag is "h6". NOTE: All tags need to be closed by a "/' mark.

2. Optional: To center text use the "center" tag

3. Add Your Content

The "hr" tag helps break content apart from others through creating a line of separation from the rest of the content.

1. For the first section of your website use a "h3" tag for your title.

2. For your more lengthy paragraphs use the "p" tag to display text onto your screen.

3. Note: to center text you can create a class called "center" which will center all your text for you in CSS. To create a class in css use a "." and then name the class. Place all of the class content in "{}" brackets.

Extra: You can also change the text color of your titles and paragraphs. In CSS, put the name of the tag you're using and then inside the brackets put color: "purple". To change the font you're using include in the brackets font-family: "'Times New Roman', serif". This allows for you to create a font family which allows you to prioritize a list of fonts you would like to use. For example if Times New Roman cannot be accessed the text would be changed to serif.

4. Adding Images

In HTML images are defined using a "img" tag. Use img src= " " to specify a URL in which you want to access a photo from. The "alt" tag is used a a description of your picture. This is helpful if your website doesnt load the picture, the user will still be able to know what the image was supposed to be.

Note: You cannot just copy the image url. Right click and copy the image address and then paste that for the image src.

5. Adding Videos

The tag iframe is used when you want to display another web page in your web page such as a youtube video.

1. Use the format of "iframe src="youtube-video"

2. You can also change the height and width of a image or video by adding within the iframe tag "height=num" and "width=num"

6. Creating "DIVS"

A div element creates a section or division. You can identify a div by giving it a class name.

1. For example: To center a section of elements create a class in your css file. To create a class for a div use "div.nameofclass" and then brackets

2. For the purpose of this lesson create a class that will center the text of your div elements. To center text use "text-align:center;
